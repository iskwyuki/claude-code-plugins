# セキュリティ関連ファイル編集時の注意事項

このファイルはセキュリティに関わるコードです。以下を必ず確認してください。

## チェックリスト

- [ ] APIキー・トークン・パスワード等の機密情報がハードコードされていないか
- [ ] 環境変数経由で機密情報を取得しているか
- [ ] ユーザー入力のバリデーション・サニタイズが行われているか
- [ ] SQLインジェクション・XSS等のインジェクション攻撃に対する防御があるか
- [ ] 認証・認可のチェックが適切に実装されているか
- [ ] エラーメッセージに機密情報が含まれていないか
- [ ] ログ出力に機密情報が含まれていないか

## 変更後の確認

- 変更内容にセキュリティリスクがないか、ユーザーに報告すること
- 重大なリスクがある場合は実装前に警告すること
