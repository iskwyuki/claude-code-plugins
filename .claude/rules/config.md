# 設定ファイル編集時の影響範囲確認

設定ファイルを編集しています。変更の影響範囲を確認してください。

## 確認事項

- この設定変更がどの範囲（開発環境/ステージング/本番）に影響するか把握すること
- 他の設定ファイルとの整合性を確認すること（例: tsconfig と package.json の依存関係）
- CI/CDパイプラインへの影響を確認すること

## 変更前の確認

- 変更の影響範囲をユーザーに報告してから実施すること
- 本番環境に影響する変更は特に慎重に扱うこと
- 破壊的変更の可能性がある場合は事前に警告すること
